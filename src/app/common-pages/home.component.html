MEAN Stack Technologies

The MEAN stack is MongoDB, Express.js, AngularJS (or Angular), and Node.js. Because all components of the MEAN stack support programs are written in JavaScript, MEAN applications can be written in one language for both server-side and client-side execution environments.

  • Angular

    Learn one way to build applications with Angular and reuse your code and abilities to build apps.
  • NodeJS

    Node.js (Node) is an open source development platform for executing JavaScript code server-side.

  • MongoDB

    MongoDB is a cross-platform and open-source document-oriented database, a kind of NoSQL database.
